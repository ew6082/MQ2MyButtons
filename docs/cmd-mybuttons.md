# /mybuttons

## Syntax

<!--cmd-syntax-start-->
```eqcommand
/mybuttons [on | off | reload | reloadcolors | show | # | help]
```
<!--cmd-syntax-end-->

## Description

<!--cmd-desc-start-->
Controls the plugin with fairly self-explanatory options, # will use the button specified.
<!--cmd-desc-end-->

## Options

| Option | Description |
|--------|-------------|
| `(no option)` | Toggle the button window on/off |
| `on|off` | Turns the plugin on and off |
| `reload` | Reload hotkeys from the INI |
| `reloadcolors` | reloads the UI. This can be very slow. |
| `show` | Show buttons configuration |
| `<ButtonNumber>` | Will use the button specified. Example: `/mybuttons 1` |
| `help` | Show help |
